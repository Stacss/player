
# Player Laravel Приложение

Это Laravel-приложение с названием "Player," созданное для управления и отображения результатов пользователей. Оно включает в себя API-точки для сохранения результатов и получения топовых результатов.

 ## Стек
PHP 8.1
MySQL 5.6
Laravel 9

## Установка
Чтобы настроить Player Laravel приложение, следуйте этим шагам:

- Клонировать Репозиторий:


```
git clone https://github.com/Stacss/player.git
```

- Перейти в Директорию Проекта:

```
cd player
```

- Установить Зависимости с Composer:


```
composer install
```

Скопировать Файл Окружения:

```
cp .env.example .env
```

- Сгенерировать ключ приложения:

```
php artisan key:generate
```

- Настроить Базу Данных:

Обновите файл .env вашими данными для подключения к базе данных:


```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ваше_название_базы_данных
DB_USERNAME=ваш_пользователь_базы_данных
DB_PASSWORD=ваш_пароль_базы_данных
```

- Запустите миграции:

```
php artisan migrate
```

- Вы можете наполнить БД тестовыми данными:

```
php artisan db:seed
```

- Запустите Приложение:

Запустите встроенный сервер Laravel:

```
php artisan serve
```

Приложение теперь доступно по адресу http://localhost:8000.

### Swagger Документация:
Чтобы просмотреть Swagger документацию, перейдите по ссылке http://localhost:8000/api/documentation.

### Реализованные API-Точки:
- Сохранение Результата:

POST /save-result

Эта точка API используется для сохранения результатов игры.

- Получение Топовых Результатов:

GET /top-results

Получите топовые результаты игры.
